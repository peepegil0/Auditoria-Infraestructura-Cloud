# Auditoría de Infraestructura Cloud con XPath

## Descripción

En este proyecto realizo una auditoría sobre un archivo XML que contiene información de infraestructura cloud. Para ello utilizo **XPath**, con el objetivo de extraer datos concretos de forma eficiente sin afectar al rendimiento del sistema.

El trabajo está basado en un caso práctico donde se simula un entorno profesional de auditoría de sistemas.

## Contenido del repositorio

* **CatalogoCloud.xml** → Archivo con los datos de la infraestructura
* **CatalogoCloud.xsd** → Esquema de validación del XML
* **auditoria_apellidos_nombre.xbook** → Archivo con las consultas XPath realizadas
* **README.md** → Explicación del proyecto

## Herramientas utilizadas

* Visual Studio Code
* Extensión **XPath Notebook**
* Lenguaje **XPath**

## Objetivo

El **objetivo** de esta práctica es aprender a:

* Navegar por documentos XML
* Utilizar expresiones XPath
* Filtrar información mediante condiciones
* Obtener datos específicos de forma eficiente

## Misiones realizadas

### Misión 1: Mapeo de Seguridad

He extraído los puertos de los servicios que se ejecutan en el centro de datos de París utilizando rutas XPath absolutas y relativas.

### Misión 2: Auditoría de Mantenimiento

He obtenido la versión del sistema operativo de un servidor concreto filtrando por su identificador.

### Misión 3: Inventario de Alta Capacidad

He recuperado los discos con capacidad igual o superior a 8 TB utilizando predicados matemáticos.

### Misión 4: Eficiencia Energética

He identificado el primer servidor apagado dentro del sistema usando funciones de XPath.

### Misión 5: Desafío del Auditor Senior

He extraído la arquitectura de CPU de los servidores que tienen GPU mediante navegación inversa.
